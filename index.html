Bookmarklets for Accessibility Testing

Bookmarklets for Accessibility Testing use JavaScript to highlight roles, states, and properties of accessibility elements on the page. They are accessible to screen reader users and work on any browser including mobile phones.

Drag and drop the (JavaScript Bookmarklet) link enclosed in (parentheses) into your bookmarks bar to use in any browser. The first link in each list item is to the bookmarklet demonstration page.

Bookmarklets now support accessibility testing of same-domain <iframe> elements but not cross-domain iframes.

  1. Forms (Forms)
  2. Headings (Headings)
  3. Images (Images)
  4. ARIA (ARIA)
  5. Tables (Tables)
  6. Lists (Lists)
  7. Landmarks (Landmarks)
  8. Meta Viewport Content (Meta Zoom)
  9. Page Title + Language (Page <title>+lang)
  10. Tabindex (Tabindex)
  11. Title Attributes (Title attribute)
  12. Iframes (<iframe>s)
  13. Force Focus (Force Focus)
  14. Reset Page (Reset Page)
  15. Landmark Heading Keyboard Navigation (landmark-heading-nav)
  16. (Grayscale)
