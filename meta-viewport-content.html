Bookmarklets for Accessibility Testing: Headings, Images, Forms, Page Title + Language, (more soon)

Meta Viewport Content Bookmarklet for Accessibility Testing displays a JavaScript alert() dialog with the <meta name=viewport> content attribute so you can tell if pinch-to-zoom has been disabled. If user-scalable=0 then mobile users cannot increase the size of the web page with the standard pinch-to-zoom gesture.

http://pauljadam.com/bookmarklets/meta-viewport-content.html

Installation

Meta Viewport Content Bookmarklet (drag link to bookmarks bar)

Not so easy installation method:

  1. On iOS Mobile Safari tap once on the textarea element below containing the JavaScript Bookmarklet URL Code.
  2. Tap Select All.
  3. Tap Copy.
  4. Add this page as a bookmark to your mobile safari browser with the name you want to call this Bookmarklet.
  5. Go to your bookmarks and naviate to the newly added bookmark.
  6. Edit the bookmark you just created.
  7. Delete the full http URL.
  8. Paste the JavaScript Bookmarklet URL Code.
  9. Save the updated bookmark.
  10. Visit the page you want to test for accessibility problems.
  11. Activate your newly created bookmarklet.
